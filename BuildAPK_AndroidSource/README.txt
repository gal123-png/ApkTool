
========================================
BuildAPK - Android Source Code
========================================

CARA BUILD (ANDROID STUDIO):
1. Buka Android Studio di Laptop.
2. Pilih "Open" atau "Import Project".
3. Arahkan ke folder "BuildAPK_AndroidSource" ini.
4. Tunggu proses Sync Gradle selesai.
5. Sambungkan HP Android atau gunakan Emulator.
6. Klik tombol "Run" (Segitiga Hijau).

CARA BUILD (COMMAND LINE):
1. Pastikan Java (JDK 17) dan Android SDK terinstall.
2. Buka terminal di folder ini.
3. Jalankan: gradle buildDebug
